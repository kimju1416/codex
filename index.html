AI 어시스턴트
대화·검색·글쓰기·문서 분석·프레젠테이션까지 — 가장 기본이 되는 AI
🟣
Claude.ai
Anthropic · claude.ai
글쓰기·분석·코딩에서 최상위 성능을 보이는 AI. 블라인드 테스트 글쓰기 품질 1위에 자연스러운 한국어 작문이 강점이에요. 최신 Fable 5 모델(6.9 출시)은 Opus를 뛰어넘는 새 티어지만 6.12 미정부 수출 통제로 현재 접근 불가. 200K 토큰 컨텍스트로 긴 문서도 한 번에 처리하고, 환각(잘못된 정보 생성)이 경쟁 AI 중 가장 낮아요. Projects로 프로젝트별 맞춤 AI 공간을 만들고, Artifacts로 대화 중 코드·문서·차트를 실시간 미리보기 가능. MCP 연동으로 Gmail·캘린더·Drive까지 직접 연결.
✅보고서·기획서·블로그·이메일 초안 작성
✅PDF·엑셀 업로드 후 내용 기반 Q&A
✅웹 검색 내장 — 최신 정보를 실시간으로 반영
✅긴 문서 요약·번역·감수 — "이 계약서 핵심 3줄로 요약해줘"
✅MCP 연동 (Gmail·캘린더·Drive 등)
⚠️이미지 생성은 불가 — 이미지가 필요하면 ChatGPT나 Midjourney 사용
글쓰기분석번역문서작성MCP무료티어
🟢
ChatGPT
OpenAI · chatgpt.com
전 세계 사용자 수 1위, 가장 범용적인 올인원 AI. 대화·이미지 생성(GPT Image)·음성 대화·Deep Research·코드 실행·파일 분석을 하나의 앱에서 모두 처리해요. Custom GPT를 만들어 반복 작업을 자동화할 수 있고, Memory 기능으로 대화 맥락을 기억. "이 문서 읽고 요약한 뒤 인포그래픽으로 만들어줘"처럼 여러 작업을 한 흐름에서 연결.
✅대화하다 바로 이미지 생성 — "카페 오픈 포스터 만들어줘"
✅Deep Research — "최근 AI 활용 사례 논문 10개 정리해줘"
✅엑셀 업로드 후 "이 매출 데이터 분석해줘" → 차트+해석
✅음성 대화 — 영어 회화 연습 상대로 활용 가능
⚠️무료 버전은 기능 제한. Plus($20/월) 구독 시 풀 기능
올인원이미지음성 대화Deep Research무료티어

Gemini
Google DeepMind · gemini.google.com
Google 생태계와 가장 깊이 통합된 AI. Gmail·Docs·Drive·캘린더·YouTube와 직접 연동. 100만 토큰 컨텍스트로 긴 문서도 한 번에 처리 가능. 응답 속도가 빠르고 멀티모달 이해력이 뛰어나요. Google Workspace 사용자라면 가장 자연스럽게 쓸 수 있는 AI.
✅Gmail에서 "이 메일 요약해줘" → 답장 초안까지 생성
✅Google Docs에서 "이 문단 더 쉽게 바꿔줘" 직접 편집
✅유튜브 영상 URL 붙여넣기 → 내용 요약·핵심 추출
✅Google 검색 결합 — "이번 주 마케팅 트렌드 정리해줘"
⚠️한국어 글쓰기 품질은 Claude보다 약간 뒤처짐
Google 연동GmailDocs무료티어100만토큰

Perplexity
Perplexity AI · perplexity.ai
검색과 AI를 결합한 "답변형 검색 엔진". 모든 답변에 출처 번호가 붙어서 팩트 확인이 바로 돼요. 실시간 웹 검색 결과를 반영. AI 전용 브라우저 Comet도 출시돼서 브라우저 안에서 AI가 항공권 예약·양식 작성 같은 작업을 직접 처리. Pro 모드에서는 여러 소스를 교차 검증해서 깊이 있는 리서치도 가능.
✅"2026 노동법 개정 내용 알려줘" → 출처 링크 포함 답변
✅"AI 생산성 최근 연구 논문 찾아줘" → 학술 검색
✅Comet 브라우저로 웹사이트 내용 자동 요약·비교
✅무료로 하루 여러 번 사용 가능. Pro($20/월)에서 무제한
⚠️글쓰기·분석은 Claude가 낫고, 검색·팩트 확인은 여기가 최고
실시간 검색출처 제시Comet팩트체크무료티어

Genspark
Genspark AI · genspark.ai
GPT·Claude·Gemini 등 여러 AI를 한 플랫폼에서 쓸 수 있는 올인원 AI. 80개 이상의 내장 도구로 채팅·PPT 생성·이미지·영상·딥리서치까지 하나의 구독으로 해결해요. 같은 질문을 여러 AI에 동시에 던져 답변을 나란히 비교할 수 있어서 "어떤 AI가 이 작업에 가장 잘 맞는지" 직접 확인하기 좋아요.
✅여러 AI 답변을 동시에 비교
✅채팅하다 바로 PPT·이미지 생성
✅"AI 활용 사례 조사해줘" → 딥리서치로 수백 소스 정리
✅하나의 구독으로 ChatGPT·Claude·Gemini 모두 사용
⚠️각 기능 품질은 전문 도구(Midjourney, Claude 등)보다 얕음
멀티AI올인원PPT생성딥리서치무료티어

Microsoft Copilot
Microsoft · copilot.microsoft.com
Windows 11·Edge 브라우저·Office 365에 내장된 AI. 별도 앱 설치 없이 Windows 작업표시줄에서 바로 대화 가능. Word에서 문서 초안 작성, Excel에서 데이터 분석·수식 제안, Teams에서 회의 자동 요약까지 AI가 처리. 이미 Microsoft 생태계를 쓰고 있다면 가장 자연스럽게 AI를 시작할 수 있는 선택.
✅Word에서 "프로젝트 제안서 초안 만들어줘" → 문서 자동 생성

✅Teams 회의 자동 요약·회의록 생성
✅Windows 기본 탑재 — 설치 없이 바로 사용

예술적 감성과 스타일 표현에서 독보적인 이미지 AI. 수채화·유화·영화 스틸·일러스트·3D 렌더링 등 어떤 스타일이든 최고 품질로 뽑아줘요. v7의 Omni Reference 기능으로 같은 캐릭터·인물을 여러 이미지에 일관되게 유지 가능. 웹 에디터에서 부분 수정(인페인팅)도 가능해져서 "이 부분만 바꿔줘"가 돼요. 이미지 안 텍스트 렌더링은 약한 편.
✅"watercolor painting of a spring park with cherry blossoms" → 감성 포스터
✅앨범 표지·이벤트 배경·콘셉트 아트 제작
✅아트 레퍼런스 — "모네 스타일로 바다 풍경"
✅웹 인터페이스에서 바로 생성 (Discord 없이도 가능)
⚠️무료 플랜 없음. Basic $10/월부터. 이미지 안 텍스트는 약함
감성·예술스타일일러스트웹 에디터

Veo 3
Google DeepMind · Gemini / Google Vids
영상과 음성·효과음·대사를 동시에 생성하는 AI의 원조. "바닷가 노을을 보는 소년, 파도 소리와 갈매기 울음"을 입력하면 영상+소리가 한 번에 완성. 대사·효과음·배경음까지 한 번에 생성하는 완성도가 가장 높아요. 8초 기본, 확장하면 더 긴 영상도 가능. 물리 법칙 이해가 뛰어나서 자연스러운 움직임 표현이 강점. 현재 최신 버전은 Veo 3.1.
✅"카페에서 바리스타가 커피 내리는 장면, 잔잔한 재즈" → 영상+소리
✅SNS·유튜브용 — "우주에서 본 지구, 웅장한 배경음악"
✅Midjourney 이미지 → Veo로 영상 변환 (이미지→영상)
✅Google Vids에서 무료 계정도 월 10회 생성 가능
⚠️풀 기능은 Google AI Ultra 구독 필요

Kling 3.0
Kuaishou · klingai.com
텍스트→영상, 이미지→영상 모두 가능한 가성비 최고 영상 AI. Midjourney 이미지를 넣으면 자연스럽게 움직이는 영상으로 변환. 4K/60fps 고화질 지원, 5개 언어 립싱크, 최대 6컷 멀티샷 스토리보드로 장면 연결이 가능해요. 음성도 동시 생성(Omni Native Audio) 가능. 전 세계 6천만+ 크리에이터가 사용 중이고, 매일 무료 크레딧(66크레딧/일, 약 3~6개 클립) 제공.
✅Motion Brush — 이미지에서 움직일 부분만 브러시로 지정
✅SNS 숏폼·프레젠테이션용 영상 클립
✅"도시 거리를 달리는 사람" → 텍스트만으로 영상 생성
✅가성비 최고 — 무료 크레딧으로 시작
⚠️무료는 720p+워터마크, 상업적 사용 불가. 워터마크 제거는 Standard $6.99/월~
이미지→영상4K멀티샷무료크레딧
